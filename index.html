The HIMYM final

The finale of the series 'How I met your mother' split the fans - some were outraged that the mother died, others were happy that Ted finally got his Robin and yet others were just sad that the series ended . Here, we seek to investigate these sentiments expressed on the interwebs whilst the last season aired, by the use of networks theory and sentiment analysis.

Intro to data

The dataset investigated is tweets tweeted in the timeperiod : 01.03.2014-03.04.2014 containing #HIMYM. In this timeperiod the last 6 episodes of the final season aired in the US, with the final episode airing on 31.03.2014. These old tweets can be fetched by using this project or by simply downloading this xlsx file. To be able to actually work with this, the data must first be cleaned, as described in the jupyter notebook that can be downloaded in the topbanner.

This results in a dataset of 265.625 tweets!

The data contains the tweet-text, username, hashtags, date and time. A few examples of what was tweeted in the given timeperiod can be seen below:

Sentiment analysis

A sentiment/average happiness value can be computed for each tweet, by looking up each word in the LabMT wordlist (must be converted to a .txt file before working with it) and averaging over the values found in the list.

As it is a bit difficult to visualize the sentiment values for 265.625 tweets, we instead compute an average sentiment value for all tweets tweeted in the same hour, leaving us with 793 average sentiment values. These are plotted on a timeline below, where the vertical dashed lines describes the releases of new episodes. 

The red circles indicates certain events that results in a big peak in sentiment, which are further described below:

Network

We wish to deeper investigate what was expressed on the interwebs whilst the final season of HIMYM aired. To do this we create a graph, where the nodes are the twitter-users. Nodes are connected if the users has created tweets with similar sentiment-value and with at least one of the same hashtags (note that the dataset has 19.620 different hashtags):

This graph is created in the hope that it will contain communities, that can be used to create wordclouds that characterize these communities -  and hopefully these wordclouds will give a more nuanced image of how these users expressed their sad-, mad- or happiness.

PreviousNext

We note that 37 communities are found. It is chosen to only investigate those of special interest (e.g. communities with a high/low average sentiment-value or with high occurence of certain hashtags).

Wordclouds are created for every community by computing TF-IDF values for each word. TF (term frequency) describes the number of times a word occurs in a community. IDF (inverse document frequency) is a factor that diminishes the weight of words that occur very frequently in the communities, and increases the weight of terms that occur rarely. And therefore TF-IDF can find words that uniquely describes the community, such that we can plot then in a wordcloud, where words with higher TF-IDF values appear bigger.

To make TF and IDF more tangible, we below show 2 wordclouds, where one is created only based on the TF-values and the other based on the IDF-values: 

TF

The words that occur the most in the given community.

IDF

The words in the community, that occur more rarely in the other communities.

Where we note that TF creates words that are not very specific for a given community, as these words occur in most of them, whilst IDF does the opposite.
